Basic Git Commands
-> git config --global user.email "Your Email Address"
-> git config --global user.name "Your Name"
-> git init => initialize git to the repositry
-> git add . => add all the files to your repositry
-> git add ./file_name => add certain files (with file_name) to your repositry
-> git commit -m "Message to be added"
-> git remote add origin url => to access repositry in system
-> git status => to check the status
-> git log => to check previous history 
-> git push -m origin master
